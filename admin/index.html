KTB Media CMS

Ten panel działa przez Netlify Identity + Git Gateway. Po kliknięciu zalogowania powinno otworzyć się okno logowania albo formularz ustawienia hasła z linku zaproszenia.

Sprawdzam Identity…

Jeśli otworzyłeś link z maila i widzisz token w adresie, ten ekran powinien obsłużyć go automatycznie. Gdyby nie poszło, wejdź bezpośrednio na /admin/index.html.
